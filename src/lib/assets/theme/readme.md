# How to use the theme

## Types of templates

- cover
- card
- index
- image

## Types of subtemplates

- presentation
- magic

### Information to fill

#### First slide

- theme (number of the current theme, this number will be shown on the full theme)

#### General

- template (type of template to use)
- subtemplate (type of subtemplate to use)

#### Cover

- title
- description
- coverImg (image to use on the cover)

#### Card

- presentation-img
- presentation-title
- presentation-description
- presentation-icons {icon, title}
